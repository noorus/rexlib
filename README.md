rexlib
======

An open library for interfacing [REX audio loop files](http://www.propellerheads.se/products/recycle/index.cfm?fuseaction=get_article&article=rex) using the [REX Shared Library](http://www.propellerheads.se/download/index.cfm?fuseaction=get_article&article=rexsharedlibrary/) from Propellerhead.
